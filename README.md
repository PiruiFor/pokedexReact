## Configuración

1. Clona este repositorio en tu máquina local:

```bash
git clone https://github.com/Emojins/Pokedex-React.git
```

2. Instala las dependencias del proyecto:

```bash
cd nombre-del-repo
npm install
```

3. Inicia la aplicación Pokedex:

```bash
npm run dev
```
